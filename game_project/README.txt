JOGO PYGAME - AVENTURA PIXEL

Como executar:
1. Instale o Python 3.
2. Instale o Pygame:
   pip install pygame
3. Entre na pasta do projeto e rode:
   python main.py

Controles:
- A/D ou setas: mover
- Espaço, W ou seta para cima: pular
- R: reiniciar
- ESC: sair

Observações:
- Os frames foram reorganizados em pastas por animação.
- O fundo é desenhado pelo próprio código para evitar depender de imagem externa.
- O jogo já inclui plataformas, inimigos, dano, espinhos e bandeira de chegada.
